
BUBBLE.PRG is the original submitted version

BUBBLEV2.PRG is an alternate version that
adjust color palette over time.

BUBBLEV3.PRG is an alternate version that
uses PLAY.PRG to play background music
while the bubble-universe is drawing.

BUBBLEVP8.PRG is a Prog8 port of the
original BASIC version.  It has a slight
speed increase.
