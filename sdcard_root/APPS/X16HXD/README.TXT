
X16HXD IS SIMILAR TO THE WINDOWS UTILITY HXD

IT IS A HEX EDITOR, BUT EXCLUSIVELY FOR USE
ON EDITING FILES (NOT MEMORY).

PRESENTLY IT IS LIMITED TO 20K OF CONTIGUOUS
MEMORY.

USE SAMPLE.DAT AS AN INITIAL SAMPLE OF
USING THE PROGRAM AND VIEWING OR EDITING
FILE CONTENT.

THE CC65 C SOURCE CODE IS AVAILABLE HERE:
https://github.com/voidstar78/X16HXD
